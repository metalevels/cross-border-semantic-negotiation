# Quick Start Guide

## Installation

```bash
git clone https://github.com/metalevels/cross-border-semantic-negotiation.git
cd cross-border-semantic-negotiation
pip install -e ".[dev]"
```

## Run the Demo

```bash
make demo
```

## Run Tests

```bash
make test
```

See `examples/italy_germany_demo.py` for a complete example.
